Create a resource map

If you are creating a new data package. You must create a resource_map. Resource_maps provide information about the resources in the data package (e.g.Â what data files should be included in the package, where is the metadata, etc.)

To create a new resource map with an existing published metadata_pid and data_pids, use the following command.

resource_map_pid <- arcticdatautils::create_resource_map(mn,
                                                         metadata_pid = metadata_pid,
                                                         data_pids = data_pids)
